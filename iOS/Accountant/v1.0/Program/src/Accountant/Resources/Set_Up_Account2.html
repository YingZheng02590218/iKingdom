3. 勘定科目を設定しよう

* 勘定科目の確認

勘定科目毎にスイッチがあります。
使用したい勘定科目をONに、不使用の勘定科目をOFFに設定してください。
表示方法は2通りあります

* 勘定科目を一覧で表示

勘定科目 > 勘定科目一覧 > 勘定科目一覧画面

表示順：流動資産 → 固定資産 → 繰延資産 → 流動負債 → 固定負債 → 資本 → 売上 → 販売費及び一般管理費 → 営業外損益 → 特別損益 → 税金

* 表示科目別に勘定科目を表示

勘定科目 > 表示科目別勘定科目一覧 > BS/PL切替えボタン

BS (貸借対照表)科目とPL (損益計算書)科目の表示を切り替えることができます。
